## Scripts

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:3000
yarn start

# build for staging
yarn stage

# build for production
yarn build

# serve production build
yarn serve

# format using prettier
yarn format
```

## Installation

1. Run `yarn install`
2. Run `yarn start`
3. Go to http://localhost:3000/
